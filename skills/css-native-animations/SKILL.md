Ao avaliar se uma animação pode ser resolvida sem JavaScript, invoque a skill global `css-native-animations` (via Skill tool) para os padrões de Scroll-Driven Animations nativas, View Transitions API, `@starting-style` e Container Queries. Usar como alternativa mais leve ao GSAP/Framer Motion quando o efeito não exige controle fino de timeline — sempre respeitando a regra de animar apenas `transform`/`opacity`.
