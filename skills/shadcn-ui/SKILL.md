Ao usar componentes shadcn/ui como base (formulários, modais, menus), invoque a skill global `shadcn-ui` (via Skill tool) para descoberta e instalação de componentes. Nunca usar shadcn/ui no estado genérico padrão — sempre customizar raio, cor e sombra para bater com o `design-system.md` do cliente (regra do `high-end-visual-design`: "NEVER in its generic default state").
