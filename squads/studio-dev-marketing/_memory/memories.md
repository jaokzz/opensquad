# Squad Memory: Studio Dev Marketing

## Estilo de Escrita

- Usar "você" singular, nunca "vocês" — fala direta com uma pessoa
- Parágrafos de legenda: máximo 2 linhas cada — blocos longos não são lidos no mobile
- Números específicos sempre: "180% mais contatos" não "muito mais contatos"
- Frases curtas no copy: máximo 10 palavras por frase em hooks
- Sem superlativas vazias: "incrível", "revolucionário", "viral" são proibidos sem prova
- Linguagem do cliente, não do desenvolvedor: "mais clientes" não "taxa de conversão"

## Design Visual

- Paleta oficial: #F2F0EC (fundo), #0C0A07 (texto), #E8A020 (destaque âmbar), #E8C547 (âmbar claro)
- Fontes: DM Serif Display italic para headlines — Plus Jakarta Sans weight 500+ para corpo
- Double-bezel obrigatório em cards (outer shell #0C0A0706 + inner white #FFFFFF)
- Dimensões: 1080×1920 Reel | 1080×1440 Carrossel
- Font floor: Hero ≥ 94px | Heading ≥ 72px | Body ≥ 34px | Caption ≥ 26px
- Proibido: Inter, Roboto, gradiente roxo/azul, texto < 20px, slide counter no HTML

## Estrutura de Conteúdo

- Hook nos primeiros 0-2s: obrigatório, sem apresentações ("Oi sou o João" = rejeição automática)
- CTA único e específico: "Comenta SITE" é válido — "Curte e compartilha" é genérico demais
- Loop design no final de Reels: último frame conecta ao início para incentivar replay
- 5 ângulos distintos por tema: cada um com driver emocional diferente (nunca 5 variações do mesmo gancho)
- Legenda: primeiros 125 chars funcionam como hook independente (antes do "...mais")
- Máximo 15 hashtags — relevância sobre quantidade

## Proibições Explícitas

- Nunca: "Did you know" / "Você sabia que" — abertura clichê, sinal de conteúdo genérico
- Nunca: "In this post I will" / "Nesse vídeo vou" — desperdiça os 2 primeiros segundos
- Nunca: CTA duplo no mesmo post — um Reel, uma ação
- Nunca: Inter ou Roboto como fonte principal — viola identidade visual
- Nunca: gradiente roxo ou azul — estética saturada de IA gerada
- Nunca: "seguro o suficiente" em segurança — binário: resiste ou não resiste
- Nunca: recomendar WordPress para sites com animações complexas
