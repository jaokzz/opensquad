# Research Brief — Studio Dev Marketing

## Fontes e Frameworks de Domínio

### Animações Web
- **Motion (ex-Framer Motion)**: Mais rápido que GSAP em 2026, melhor integração React/Next.js, tree-shaking nativo. Fonte: motion.dev, logrocket.com/best-react-animation-libraries
- **GSAP**: Padrão profissional para scroll complexo, Disney/Google/Apple o usam. Ideal quando precisão de timeline importa mais que DX. Fonte: greensock.com
- **Three.js + R3F**: Para WebGL/3D. React Three Fiber dá API declarativa. Fonte: threejs.org, docs.pmnd.rs/react-three-fiber

### Instagram Reels
- Contas com 2+ Reels/semana crescem 4.2x mais rápido que contas apenas com feed posts. Fonte: truefuturemedia.com
- 80% do desempenho de um Reel é determinado pelo hook (primeiros 3s). Fonte: flowshorts.app/blog/instagram-guide
- Algoritmo 2026 pesa: shares para DMs > saves > comments > likes. Fonte: influenceflow.io
- "Originality Score" penaliza conteúdo reciclado do TikTok. Fonte: influenceflow.io
- Trial Reels: mostrar apenas para não-seguidores (feature 2024-2026). Fonte: jumpfly.com/blog
- Duração ideal para completion rate máximo: 15-30s. Áudio original recebe boost algorítmico. Fonte: influenceflow.io

### Segurança Web
- PMEs alvo de 3x mais ataques que grandes empresas. Custo médio de breach: $200K. Fonte: cokoagency.com/blog
- 60%+ dos ataques exploram vulnerabilidades em software desatualizado. Fonte: pintox.com/web-development-security-checklist
- Checklist 2026: SSL/HTTPS, MFA, WAF, patches, backups — 5 medidas base. Fonte: capitalnumbers.com/blog/web-security-best-practices-2026
- Bots agentic (IA) tomando decisões em tempo real: nova ameaça 2026. Fonte: hosting.com/blog/website-security-checklist
- 40% dos servidores MCP têm falhas de segurança. Fonte: perimattic.com/web-development-security-checklist

## Vocabulário Profissional

### Web Development
- **Sempre usar**: LCP, CLS, Core Web Vitals, will-change, GPU-accelerated, ScrollTrigger, cubic-bezier, React Three Fiber, tree-shaking
- **Nunca usar**: "site bonito" (sem métrica), "animações suaves" (vago), "moderno" (sem contexto técnico)

### Instagram Marketing
- **Sempre usar**: hook, ângulo, completion rate, replays, shares para DM, originality score, CTA específico, fold (ponto de corte da legenda)
- **Nunca usar**: "conteúdo de qualidade" (vago), "engajamento" sem especificar métrica, "viral" como objetivo

### Segurança
- **Sempre usar**: OWASP Top 10, CVE, CVSS score, supply chain, threat model, attack surface
- **Nunca usar**: "seguro o suficiente", "provavelmente não vai ser atacado"
