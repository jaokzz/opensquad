# Task: Orchestrate Custom Request

## Objetivo
Para qualquer pedido que não siga o pipeline padrão de Reel, Marco analisa, monta o fluxo ideal, e coordena os agentes necessários.

## Processo

1. **Analisar o pedido**: identificar objetivo final, entregáveis esperados, e quais agentes têm as capacidades necessárias
2. **Montar fluxo personalizado**: sequência de agentes, dependências, pontos de checkpoint
3. **Apresentar plano** antes de executar (máximo 5 linhas)
4. **Executar com delegação contextual**: cada agente recebe o contexto exato que precisa
5. **Sintetizar resultado**: avaliação crítica + próximos passos recomendados

## Exemplos de Pedidos Customizados

- "Quero uma campanha de lançamento de site" → site (André) + auditoria (Sandro) + conteúdo de lançamento (Lucas+Gabriela+Daniela) + prospecção de leads (Pedro)
- "Audita meu site e cria conteúdo sobre os problemas que encontrar" → Sandro → Lucas → Gabriela → Caio
- "Preciso abordar 3 leads essa semana" → Pedro (qualifica e scripts para os 3) → checkpoint
- "Monta estratégia completa para o mês" → Lucas (calendário estratégico) → André (se precisar de site) → Gabriela+Daniela (conteúdo) → Pedro (prospecção paralela)
