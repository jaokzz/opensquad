# Anti-Patterns — Visual Prompt Lab

## Prompts

### Nunca fazer
- Escrever prompt em português (resultado 40-60% pior)
- Prompt sem lista de negativos
- Adjetivos genéricos: "beautiful", "modern", "elegant" sem qualificação
- Prompt sem enquadramento definido
- Múltiplos sujeitos principais com igual importância
- Gastar crédito FLUX Pro para testar conceito não aprovado
- Alterar prompt sem passar pela Priya

## Conceitos Visuais

### Nunca fazer
- Entregar conceito único sem opções
- Clichês de banco de imagens:
  - Médico sorrindo com estetoscópio
  - Aperto de mão de negócios
  - Família feliz em frente à casa
  - Equipe sorrindo ao redor de laptop
  - Mulher com salada sorrindo
- "Estilo moderno" sem especificar o que é moderno para aquele setor
- Paleta vaga: "tons neutros", "cores vibrantes" — sempre HEX ou nomes descritivos precisos
- Avançar para Priya sem direção aprovada

## Execução

### Nunca fazer
- Gerar sem checkpoint de crédito aprovado pelo Orion
- Pular revisão da Vera
- Entregar imagem com artefatos detectáveis (mãos deformadas, texto inventado)
- Reiterar geração sem diagnóstico de qual aspecto falhou
- Forçar mais de 2 iterações no mesmo problema sem escalar para João Vitor

## Entrega

### Nunca fazer
- Entregar imagem sem instrução de uso no site
- Entregar sem nome sugerido para o arquivo
- Aprovar com ressalva vaga que vai causar problema depois
- Gerar imagem para seção sem saber o aspect ratio necessário
