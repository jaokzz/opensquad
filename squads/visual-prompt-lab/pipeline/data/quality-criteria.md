# Critérios de Qualidade — Visual Prompt Lab

## As 5 Dimensões de Aprovação (Vera Visual)

### 1. Relevância
A imagem comunica o que o negócio/marca precisa comunicar para o público certo?
- **APROVADO**: a imagem reforça o posicionamento do cliente
- **REJEITAR**: a imagem poderia ser de qualquer negócio genérico do mesmo setor

### 2. Composição
O focal point é claro? A hierarquia visual está estabelecida?
- **APROVADO**: olhar vai naturalmente para o elemento principal
- **REJEITAR**: múltiplos pontos de atenção sem hierarquia, composição sem intenção

### 3. Qualidade Técnica
A imagem está livre de artefatos de geração IA?
- **APROVADO**: nitidez correta, mãos/dedos corretos, sem texto inventado, sem deformações
- **REJEITAR**: qualquer um dos problemas abaixo presente

**Problemas técnicos comuns a verificar:**
- [ ] Mãos ou dedos deformados (dedos a mais, fusão de dedos)
- [ ] Texto inventado ou ilegível em placas/embalagens
- [ ] Olhos assimétricos, sem foco ou não-humanos
- [ ] Artefatos nas bordas ou fundo inconsistente
- [ ] Proporções corporais incorretas
- [ ] Elementos que "escapam" do enquadramento de forma estranha
- [ ] Marca d'água ou texto de geração visível

### 4. Alinhamento Criativo
A imagem reflete a direção criativa aprovada pela Iara?
- **APROVADO**: mood, paleta e estilo batem com o que foi descrito
- **REJEITAR**: a imagem é tecnicamente boa mas não é o conceito que foi aprovado

### 5. Viabilidade Web
A imagem vai funcionar na seção do site para a qual foi gerada?
- **APROVADO**: aspect ratio correto, espaço para sobreposição de texto se necessário, fundo compatível com a paleta do site
- **REJEITAR**: focal point em posição que conflita com layout, fundo que se perde no design, proporção errada para a seção

## Score de Qualidade (1-10)

| Score | Significado |
|-------|-------------|
| 9-10 | Excepcional — usar diretamente sem edição |
| 7-8 | Aprovado — pode precisar de leve ajuste de cor/crop |
| 5-6 | Aprovado com ressalva — funcional mas não ideal |
| 3-4 | Rejeitar — problemas corrigíveis com ajuste de prompt |
| 1-2 | Rejeitar — falha estrutural, reavaliar direção criativa |
